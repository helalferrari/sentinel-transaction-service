Criar Transação

POST /api/transactions

Envia uma nova transação para o pipeline de processamento.

Request Body

{
    "accountId": "ACC12345",
    "amount": 250.75,
    "merchant": "Netflix"
}

Response (201 Created)

{
    "id": "e6670de1-6bf0-465d-a5eb-a2a3dbbeb6d9",
    "accountId": "ACC12345",
    "amount": 250.75,
    "merchant": "Netflix",
    "timestamp": "2026-03-11T00:39:52.817",
    "status": "PENDING"
}

Erros Comuns

  • 400 Bad Request: Campos obrigatórios faltando ou valores inválidos.
  • 500 Internal Server Error: Problemas de conexão com o broker Kafka.
