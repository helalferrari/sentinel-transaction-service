Visão Geral

O Sentinel Transaction Service é um microserviço desenvolvido com o ecossistema Spring Boot e Apache Kafka para processar transações financeiras em tempo real.

Este sistema é o ponto de entrada para transações brutas (raw), onde passam por uma validação inicial e, em seguida, são analisadas por um motor de streaming para detecção de comportamento suspeito (fraude).

Java 21 Spring Boot 3.2 Kafka Streams

Objetivos do Projeto

  • Processamento de alta vazão (throughput) via Kafka.
  • Baixa latência na detecção de fraude.
  • Segregação clara de responsabilidades através do padrão MVC.
  • Monitoramento visual facilitado pelo AKHQ.
